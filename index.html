    JavaScript (ES6) Service Logic

    Eventn microservices execute in a secure and scalable sandboxed Node.js environment. This provides freedom to not only use all JavaScript features, but also ECMAScript 2015 (ES6) features available in Node.js (v9.x) such as async/await. Additionally, Eventn also supports a set of selected NPM modules.
